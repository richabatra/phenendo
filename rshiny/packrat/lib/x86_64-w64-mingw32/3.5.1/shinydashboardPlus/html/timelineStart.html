timelineStart {shinydashboardPlus}R Documentation

AdminLTE2 timeline starting point

Description

Create a timeline starting point

Usage

timelineStart(icon = "clock-o", color = NULL)

Arguments

icon

item icon such as "clock-o", "times", ...

color

item color: see here for a list of valid colors https://adminlte.io/themes/AdminLTE/pages/UI/general.html.
